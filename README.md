**Shop-app** — современный интернет-магазин электроники с адаптивным дизайном, созданный на React + TypeScript в рамках тестового задания.

## Особенности

- **Полностью адаптивный дизайн** — корректное отображение на мобильных (≤480px), планшетах (≤768px) и десктопах
- **Интерактивный поиск** — фильтрация товаров по названию в реальном времени
- **Модальное окно** — детальная информация о товаре с возможностью закрытия по клику вне окна, крестику или клавише Esc
- **Современный UI** — чистый дизайн с анимациями и интерактивными элементами
- **Высокая производительность** — оптимизация с помощью Vite и useMemo
- **Hover-эффекты** — интерактивные карточки товаров
